# Twitter

 To see project-level documentation, go to client/server readme.md

 ## How to build and run
 1. Genereta a new Makefile:
 ```
 cmake .
 ```
 2. Build:
 ```
 make
 ```
 3. Start server:
 ```
 ./server/server_output/server 
 ```
4. Start client:
```
./client/client_output/twitter @profile 127.0.0.1 4242
```
